# rust-homology-group 

A tiny module for finding homology group and its bases.

It also has Smith normal form ([SNF](https://en.wikipedia.org/wiki/Smith_normal_form)) module usable independently.

Requires num(0.4.0~) and nalgebra(0.31.1~).
